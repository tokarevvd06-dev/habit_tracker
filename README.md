# Трекер привычек
Веб-приложение для отслеживания полезных привычек.

# Возможности
- Добавление и удаление привычек
- Отметка выполнения с подсчетом серий (streak)
- Прогресс-бар до 30 дней
- Статистика и автосброс отметок каждый день
- Данные сохраняются в localStorage

# Технологии
HTML, Tailwind CSS, JavaScript

# Запуск
Открыть index.html в браузере
